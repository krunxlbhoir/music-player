# Music Player App

A simple and interactive music player built with HTML, CSS, and JavaScript.

## Demo
https://krunxlbhoir.github.io/music-player/

## How to run locally
Open `index.html` in a web browser.

## Features
- Play & pause music
- Next & previous song controls
- Interactive progress bar to change play time
- Responsive design
- Dynamic UI updates using JavaScript

## Notes
- Play button shadow: filter: drop-shadow(0px 8px 16px rgba(201, 59, 118, 0.25));
- Song data stored in a JavaScript array (title, author, src, image)

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
